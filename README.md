### Правила датасета:
- Invoice (идентификатор заказа)
    - Начинается с "C" - отменённый заказ
    - Начинается с "A" - нет точной информации (возможно бухгалтерия или технические транзакции)

- StockCode (идентификатор товара)
    - 5 цифр + 1-2 буквы - идектификатор + цвет товара
    - другие буквенные коды - внутренние операции (не продажи)
    - Ошибка в записи кода '47503J ' (добавлен лишний пробел)

- Quantity (кол-во товара)
    - отрицательное - возврат товара

- Price (цена за единицу товара)
    - Нулевая цена - ???
    - Отрицательная цена - ???


Рекомендация для бизнес-анализа:

Для расчета выручки (Revenue), среднего чека (AOV) и LTV вам необходимо:

Удалить все записи, где UnitPrice == 0. Они не несут финансовой ценности и в основном являются шумом системы учета.
Удалить все записи с Quantity < 0, у которых нет префикса 'C' в номере инвойса. Это не возвраты покупателей, а внутренние складские операции.
Оставить записи с префиксом 'C', если вы хотите учитывать возвраты в чистой выручке, либо исключить их, если анализируете только валовые продажи.
